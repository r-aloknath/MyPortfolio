BRIEFING

This particular system is developed to automate all the works mentioned above by putting computers in work. It will ease out the hostel registration and management process like never done before. This will save a lot of time of both college staffs and students. It can handle the management works like allocating, deallocating rooms, deleting a record, searching of a students record without worry. With one click you can see the status of a student or whole hostel within few seconds. It will also provide a brief description of notices issued by college which will help the students to take actions within time.

CONCLUSION

Hostel management system is a user-friendly and customization software for student hostels. Hostel management system has been developed to manage and automate the over-all all processing of a large student hostel. Hostel management system is capable of managing hostel rooms, student records and room allocations etc. This project is very flexible software and it can be upgraded according to the individual hostel needs.
